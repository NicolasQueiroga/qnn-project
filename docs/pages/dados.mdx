# Os Dados 

Este conjunto de dados aprimorado baseia-se no conjunto "Uncovering Knowledge: A Clean Brain Tumor Dataset for Advanced Medical Research". 
Ele passou por melhorias significativas, reconhecendo as contribuições dos autores originais.

Os dados iniciais foram obtidos do conjunto de dados de ressonância magnética para classificação de tumores cerebrais,
acessível neste [link](https://www.kaggle.com/datasets/sartajbhuvaji/brain-tumor-classification-mri), e foram generosamente compartilhados por [Sartaj](https://www.kaggle.com/sartajbhuvaji).

O Dataset consiste em 3096 imagens de ressonância magnética (MRI) de tumores cerebrais,
divididas em 4 categorias: meningioma (913 imagens), glioma (901 imagens), pituitária (844 imagens), normal (438 imagens).
